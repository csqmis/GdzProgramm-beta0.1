# GdzProgramm-beta0.1
Этот скрипт — удобный помощник для быстрого поиска заданий из учебных пособий по различным предметам для 7 класса. В текущей версии он позволяет выбрать класс, предмет и задание, а затем предоставляет прямую ссылку на соответствующее задание на сайте gdz.ru.

Что делает текущая версия:
Позволяет выбрать класс (в настоящее время поддерживается только 7 класс).
Выбирает предмет (в текущей версии — только алгебра).
Показывает список номеров заданий (от 1 до 1247).
После выбора номера задания выводит прямую ссылку на задание на сайте.
Планируемое улучшение:
Проект будет расширен для поддержки всех классов и учебных предметов, а также добавления поиска по другим ресурсам и автоматической навигации.
